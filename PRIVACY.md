# Privacy Policy

Textonly X is a browser extension for X/Twitter that changes how content is displayed on x.com and twitter.com based on the user's settings.

Data collection
- Textonly X does not collect personal data.
- Textonly X does not sell user data.
- Textonly X does not transmit user data to external servers.
- Textonly X does not use analytics, tracking, or advertising services.

Data storage
- The extension stores user preferences using Chrome extension storage.
- These preferences include visual settings such as whether to hide images, videos, quotes, profile photos, emoji, and media size preferences.
- This data is stored only so the extension can remember the user's chosen settings.

Website access
- The extension runs only on x.com and twitter.com.
- Access to these sites is required so the extension can apply the user's display preferences directly on those pages.

Third parties
- The extension includes a local copy of jQuery and does not load remote code.
- The extension does not share data with third parties.

Changes
- This privacy policy may be updated if the extension's functionality changes.

Contact
- If needed, contact the extension publisher through the Chrome Web Store listing or the project repository.
